使用指南

1. 上传图片

点击上传区域或拖拽图片文件，支持同时选择多张图片。支持JPG、PNG、WEBP、BMP、GIF等常见格式。

2. 选择模式

选择"合并为单个PDF"将所有图片合并到一个PDF文件中，或选择"每张图片独立PDF"为每张图片生成单独的PDF。

3. 调整设置

根据需求调整页面尺寸、方向、图片质量、边距等参数，获得最佳的PDF输出效果。

4. 排序图片

在图片列表中可以调整图片顺序，使用上移/下移按钮或直接拖拽来重新排列。

5. 预览转换

点击"预览PDF"可以查看转换效果，确认无误后点击"开始转换"生成最终的PDF文件。

6. 下载保存

转换完成后，点击"下载PDF"将生成的PDF文件保存到本地设备。

常见问题解答

支持哪些图片格式转换？

我们的工具支持所有主流图片格式，包括JPG/JPEG、PNG、WEBP、BMP、GIF(静态)、TIFF等。无论您的图片是什么格式，都可以轻松转换为高质量的PDF文件。

合并模式和独立模式有什么区别？

合并模式会将所有上传的图片合并到一个PDF文件中，每张图片占一页；独立模式则为每张图片生成单独的PDF文件。如果您要制作相册或文档集合，建议使用合并模式；如果需要单独的PDF文件，选择独立模式。

如何调整图片在PDF中的顺序？

上传图片后，在图片列表中可以看到所有图片的缩略图。您可以使用每张图片右侧的"上移"和"下移"按钮来调整顺序，也可以直接拖拽图片来重新排列。调整后的顺序就是PDF中页面的顺序。

转换后的PDF文件大小如何？

PDF文件大小取决于原始图片的尺寸、数量和您选择的图片质量设置。高质量设置会保持最佳画质但文件较大；中等和低质量设置会适当压缩以减小文件大小。您可以根据用途选择合适的质量设置。

可以处理多少张图片？

理论上没有严格的数量限制，但为了确保良好的处理性能和用户体验，建议单次处理不超过50张图片。如果需要处理大量图片，可以分批进行转换。
