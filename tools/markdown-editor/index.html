常见问题解答

什么是Markdown？

Markdown是一种轻量级标记语言，使用简单的语法来格式化文本。它易于学习和使用，被广泛应用于技术文档、博客写作、README文件等场景。

编辑器支持哪些Markdown语法？

支持标准Markdown语法，包括标题、粗体、斜体、列表、链接、图片、代码块、引用、表格等。同时支持部分扩展语法，如删除线、任务列表等。

如何保存我的文档？

您可以使用导出功能将文档保存为.md或.html格式到本地。编辑器还会自动保存您的内容到浏览器本地存储，下次访问时会自动恢复。

是否支持图片上传？

目前支持通过URL引用图片。您可以将图片上传到图床服务，然后在Markdown中使用图片链接。我们正在开发直接上传图片的功能。

编辑器是否免费使用？
